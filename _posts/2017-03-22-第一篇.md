#### 安装必要的开发环境
```
gem install jekyll bundler
```

#### 创建新的 jekyll 博客
```
jekyll new <PATH>
```

#### jekyll 的基本使用
```
jekyll build
jekyll serve, jekyll serve --detach
```

#### 文件结构
```

├── _data
```
